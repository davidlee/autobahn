- **outcome**: observation
- **disposition**: follow_up_backlog

DR-004 §3.4 documents this as "future direction" — promote when spec-driver exports them as StrEnums. Not blocking for DE-004.

- **outcome**: minor
- **disposition**: follow_up_backlog
- **notes**: Coupling tests verify adapter import boundaries but don't yet assert the `spec_driver.orchestration` import boundary for `api/functions.py`. Low risk since the boundary is well-defined by the 3 wrapper functions, but would strengthen regression safety.
