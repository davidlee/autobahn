## 1. Summary

- **Delta**: DE-004 ÔÇö Review operations
- **Design Revision**: DR-004
- **Desired Outcome**: autobahn exposes `prime_review`, `summarize_review_outcome`, and `disposition_finding` as public API functions that delegate to `spec_driver.workflow` composed operations. Review-related enums retired from autobahn. Dead error types removed. All existing tests pass.

## 2. Context & Constraints

- **Current Behaviour**: autobahn has no review operations. Review-related enums are autobahn-owned transcriptions. `ToolInvocationError`/`ToolContractError` exist unused.
- **Target Behaviour**: 3 new public API functions wrapping spec-driver operations. Review enums are re-exports. Dead code removed.
- **Dependencies**: spec-driver DE-124 must land first (creates `spec_driver.workflow` package with composed operations)
- **Constraints**: DE-124 blocking. No implementation until `spec_driver.workflow` is importable.
